# Super Trunfo - Nivel Aventureiro

Programa em C que compara duas cartas de países.

O usuário escolhe um atributo no menu e o programa mostra qual carta venceu.

Atributos:
1 - População
2 - Área
3 - PIB
4 - Pontos Turísticos
5 - Densidade Demográfica

Regra especial:
Na densidade vence o menor valor.

## Como compilar

Usando OnlineGDB ou outro compilador C.

## Como executar

Digite o número do atributo e veja o vencedor.
